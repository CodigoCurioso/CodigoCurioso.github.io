¿Es Permutación?

Implemente un algoritmo para determinar si una cadena de texto es permutación de otra cadena de texto.

Ejemplos

 > IsUnique("abcd");
   true

 > IsUnique("abccd");
   false

Solución

Análisis

Respuesta

function IsUnique(text) {
    let result = true;
    const lettersFound = [];
    
    for(letter of text) {
        if (lettersFound[letter]) {
            result = false;
            break;
        } 
    
        lettersFound[letter] = true;
    }
    
    return result;
}
